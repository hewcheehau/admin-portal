# admin_portal

A simple web portal using flutter development 
Functionalities include:
- view all post list
- view comment by post selection

## Flutter version
- 3.10.5

## State management
- Bloc

## Api references
- https://jsonplaceholder.typicode.com

## Route management
- auto route

## File structure
main.dart
app.dart
- application (logic)
- domain (model)
- infrastructure (helper, utils)
- l10n (localization)
- presentation (ui)

### How to run
- flutter run (entry: main.dart)
